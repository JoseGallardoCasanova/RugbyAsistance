Política de Privacidad

Rugby Asistencia · Última actualización: 31/12/2025

1. Quiénes somos

Esta política describe cómo la app Rugby Asistencia maneja datos. Si tienes preguntas, contáctanos en: jose.d.gallardo.c@gmail.com.

2. Datos que la app usa o procesa

La app está diseñada para gestionar asistencia y datos del club. Dependiendo del uso, puede procesar:

  • Credenciales de inicio de sesión: usuario (por ejemplo, email) y contraseña, para autenticar acceso.
  • Datos de usuarios: nombre, rol (por ejemplo admin/entrenador) y un identificador interno.
  • Datos de jugadores y asistencia: nombre, categoría y registros de asistencia por fecha.
  • Identificador de jugador (por ejemplo RUT u otro identificador) si tu club lo utiliza para diferenciar jugadores.

No usamos estos datos para tracking publicitario (no vendemos datos ni los usamos para rastrear entre apps/sitios).

3. Cómo se usan los datos

Los datos se usan para:

  • Permitir el inicio de sesión y control de acceso.
  • Mostrar y administrar categorías, jugadores y asistencias.
  • Sincronizar registros con un backend del club.

4. Dónde se almacenan / con quién se comparten

Según la configuración actual del proyecto:

  • La app puede guardar datos localmente en el dispositivo (por ejemplo usando AsyncStorage) para mantener la sesión.
  • La app puede enviar/recibir datos hacia servicios del club mediante peticiones HTTPS (por ejemplo, un script de Google Apps Script/Google Sheets).

5. Retención y eliminación

La sesión puede almacenarse localmente en el dispositivo hasta que cierres sesión o desinstales la app.

Si necesitas eliminación/corrección de datos del backend del club, escríbenos a jose.d.gallardo.c@gmail.com indicando tu solicitud.

6. Seguridad

Buscamos proteger los datos usando comunicaciones seguras (HTTPS) y controles de acceso. Aun así, ningún sistema es 100% infalible.

7. Cambios

Podemos actualizar esta política para reflejar cambios en la app. Publicaremos la versión actualizada en esta misma URL.
